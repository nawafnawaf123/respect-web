# Respect App Website

موقع بسيط جاهز للرفع على Vercel حتى تستخدمه في Paddle Website Verification.

## الصفحات
- /
- /pricing
- /terms
- /privacy
- /refund
- /contact

## التشغيل المحلي
```bash
npm install
npm run dev
```

## الرفع على Vercel
1. ارفع هذا المجلد على GitHub.
2. افتح Vercel.
3. New Project.
4. اختر الريبو.
5. Deploy.

بعد النشر استخدم الروابط في Paddle:
- Web domain: https://your-domain.vercel.app
- Pricing page: https://your-domain.vercel.app/pricing
- Terms: https://your-domain.vercel.app/terms
- Privacy: https://your-domain.vercel.app/privacy
- Refund: https://your-domain.vercel.app/refund
